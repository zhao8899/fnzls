## 🎨 统一设计标准

### 色彩系统规范
```css
/* WaterMIS统一色彩变量 */
:root {
  /* 主品牌色 - 智慧水蓝 */
  --primary-500: #2196f3;
  --primary-600: #1976d2;
  
  /* 成功色 - 清澈水蓝 */
  --success-500: #00bcd4;
  --success-600: #00acc1;
  
  /* 警告色 - 阳光橙 */
  --warning-500: #ff9800;
  
  /* 错误色 - 珊瑚红 */
  --danger-500: #f44336;
}
```

### 间距系统规范
```css
/* 8px基础间距系统 */
:root {
  --space-1: 4px;   /* 0.25rem */
  --space-2: 8px;   /* 0.5rem */
  --space-3: 12px;  /* 0.75rem */
  --space-4: 16px;  /* 1rem */
  --space-6: 24px;  /* 1.5rem */
  --space-8: 32px;  /* 2rem */
}
```

### 字体系统规范
```css
:root {
  --font-size-xs: 12px;
  --font-size-sm: 14px;
  --font-size-base: 16px;
  --font-size-lg: 18px;
  --font-size-xl: 20px;
  --font-size-2xl: 24px;
  --font-size-3xl: 30px;
}
```
